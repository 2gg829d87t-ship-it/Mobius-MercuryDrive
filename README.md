# Mobius-MercuryDrive # Mobius-MercuryDrive — Левитирующий двигатель на основе ртути и трубы Мёбиуса

**Автор:** Рашид Азнабаев Тагирович (концепция, геометрия)

**Дата:** Июль 2026 года

---

## Описание

Левитирующий двигатель, использующий ртуть в трубе Мёбиуса для создания подъёмной силы и тяги. Не имеет механических частей, работает бесшумно, масштабируется от микро-до макро-уровня.

---

## Принцип работы

1. Ртуть помещена в герметичную трубу Мёбиуса.
2. Через ртуть пропускается электрический ток.
3. Магнитное поле закручивает поток ртути в спираль.
4. Взаимодействие тока и поля создаёт силу, поднимающую конструкцию (левитация) и двигающую её вперёд (тяга).

---

## Преимущества

- Отсутствие механических частей.
- Стабильность (ртуть гасит колебания).
- Простота управления (изменением тока).
- Масштабируемость.
- Работает в вакууме и атмосфере.

---

## Сборка

1. Изготовить трубу Мёбиуса из немагнитного материала.
2. Заполнить ртутью.
3. Разместить кольца Мёбиуса вокруг трубы.
4. Подать ток.
5. Отрегулировать ток для достижения левитации.
